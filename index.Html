Frequent Question Asked

What is HTML ?

  • HTML (Hyper text markup language) is the standard markup language for Creating web pages and web application. it defines the structure and layout of contents on a web page using a system of tags and attributes.

what are the basics of HTML?

  1. Document structure
  2. Head Section
  3. Body Section
  4. Elements and Tags
  5. Attributes
  6. Links and Anchors
  7. Images
  8. Lists
  9. Comments.

What is CSS ?

  • CSS (Cascading style sheets) is a style sheet language used for describing a presentation of a document written in HTML.It controls the appearance of elements on a web page , including layout,colors,fonts and more.
